# 설명서
[이 링크를 클릭하여](https://github.com/deltoid7/tvroom-downloader/releases/download/v1/tvroom_downloader.exe) 프로그램을 다운로드 받습니다.<br>
[ffmpeg](https://ffmpeg.org/)를 다운로드 받습니다.<br>
실행하고, 티비룸에서 보고 있던 url을 붙여 넣으세요.<br>
url이 망가져 보여도 괜찮습니다.<br>
(예시: `https://tvroom.cc/video/%EA%B8%B0%EC%83%9D%EC%B6%A9-2019/%EB%B3%B8%ED%8E%B8`)
<br>
<br>
단기간에 요청을 너무 많이 날리면 차단당합니다.<br>
ip 기반 차단이니 vpn을 쓰시거나 [cloudflare warp](https://one.one.one.one/) 사용을 권장합니다.

<br>

# 당부
티비룸에 있는 영상은 모두 저작권 침해 영상입니다.<br>
교육용, 학습용 혹은 OTT에 없을 때만 사용하세요.

또한 이렇게 오픈소스로 뿌린 이상 막히는 건 시간문제입니다.<br>
막히면 시간 날 때 업데이트하겠습니다.

또한 문제 시 해당 프로젝트는 내려갑니다.
